Projekt Laravel: Paintball Platform

Platforma do wyszukiwania i rezerwacji gier paintballowych z funkcjami społecznościowymi

Opis projektu

Aplikacja webowa wspierająca społeczność paintballową. Platforma umożliwia firmom publikowanie ofert, a graczom – rejestrację, prowadzenie blogów, korzystanie z kalendarza rezerwacji oraz wymianę doświadczeń na forum. Projekt realizowany jest jako część pracy inżynierskiej.

Technologie i narzędzia

  • Backend: Laravel 11.5, PHP 8.x, Breeze (auth), Blade
  • Frontend: HTML, CSS, Blade
  • Baza danych: MySQL
  • Testowanie: Pest
  • Kontrola wersji: Git, GitHub
  • Planowane: Docker, GitHub Actions, Prometheus/Grafana

Główne funkcjonalności

  • Rejestracja i logowanie użytkowników
  • Dodawanie i przeglądanie ogłoszeń firm paintballowych
  • Blogi użytkowników i poradniki
  • Forum społecznościowe
  • System punktowy i ranking graczy
  • Kalendarz rezerwacji terminów
  • Mapa z lokalizacjami gier

Plan rozwoju

  • Wdrożenie CI/CD z GitHub Actions
  • Konteneryzacja z Dockerem
  • Monitoring i logowanie (Prometheus + Grafana)
  • Rozszerzenie API do aplikacji mobilnej
